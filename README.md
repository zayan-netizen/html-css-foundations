# HTML & CSS Foundations

This repository contains beginner-level HTML and CSS projects created
while learning frontend web development fundamentals.

Each project focuses on clean structure, basic styling, and simple layouts
without using JavaScript or frameworks.

## Projects Included

### 1. Personal Bio Card
A simple personal bio card webpage with centered layout and basic styling.

### 2. One-Page Resume Website
A clean resume-style webpage using semantic HTML and CSS.

### 3. Eventorin Landing Page
A landing page concept for a student-focused event management platform.

## Technologies Used
- HTML5
- CSS3

## Purpose
This repository documents my learning progress in frontend development
and serves as a foundation for future web projects.
